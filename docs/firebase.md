# Deploy to Firebase instructions:

-   Build your project
```sh
    npm run build
```

-   Install Firebase tools
```sh
    npx create-react-app client --template=typescript
```

-   Login into your Fiebase acount
```sh
    firebase login
```

-   Init your project
```sh
    firebase init
```

-   Deploy your project
```sh
    firebase deploy
```

-  Or execute this command to build and deploy
```sh
    npm run deploy
```
